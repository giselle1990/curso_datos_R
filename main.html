Ejemplo Práctico: Análisis de Contactos SUACI

# Cargar datos de atención ciudadana
atencion_ciudadano <- read.csv(
  "http://cdn.buenosaires.gob.ar/datosabiertos/datasets/sistema-unico-de-atencion-ciudadana/gcba_suaci_barrios.csv"
)

# Los 5 verbos en acción con el operador pipe %>%
atencion_ciudadano %>%
  # 1. FILTER: Solo reclamos de 2015 en Palermo o Retiro
  filter(AÑO == 2015, BARRIO %in% c("PALERMO", "RETIRO")) %>%
  
  # 2. SELECT: Quedarnos con variables relevantes
  select(BARRIO, RUBRO, TIPO_PRESTACION, total) %>%
  
  # 3. MUTATE: Crear categoría de magnitud
  mutate(categoria = case_when(
    total > 100 ~ "Alto",
    total > 10  ~ "Medio",
    TRUE        ~ "Bajo"
  )) %>%
  
  # 4. GROUP_BY + SUMMARISE: Total por barrio y tipo
  group_by(BARRIO, TIPO_PRESTACION) %>%
  summarise(total_contactos = sum(total), .groups = "drop") %>%
  
  # 5. ARRANGE: Ordenar descendente
  arrange(desc(total_contactos))
🎯

El Operador Pipe: %>%

Significa "y luego". Permite encadenar operaciones de forma legible. x %>% f(y) es equivalente a f(x, y)

Principios para Visualizaciones Efectivas

  • [1] Normalizar: Mostrar tasas per cápita en lugar de valores absolutos cuando se comparan territorios de diferente tamaño
  • [2] Contextualizar: Siempre incluir título descriptivo, fuente de datos, y unidades de medida
  • [3] Simplificar: Eliminar elementos decorativos que no aporten información (chartjunk)
  • [4] Accesibilidad: Considerar daltonismo usando paletas como viridis en lugar de rainbow

Modelado Estadístico

REGRESIÓN_LINEAL//LM()

El modelado cuantifica relaciones: permite pasar de "la luz solar mejora el crecimiento" a "por cada hora adicional de exposición, el rendimiento aumenta 1%".

📉

Simple

Una variable predictora

y ~ x
📊

Múltiple

Múltiples predictores

y ~ x1 + x2
📈

Categórica

Variables cualitativas

y ~ factor(x)

Caso: Expectativa de Vida en Argentina

# Datos de Gapminder - Argentina
data_arg <- data_mundial %>%
  filter(pais == "Argentina")

# Modelo lineal: expectativa de vida vs año
modelo_exp <- lm(expVida ~ año, data = data_arg)

# Resultado
summary(modelo_exp)

# Coeficientes:
# (Intercept)  -389.6063
# año             0.2317  ***

# Interpretación: por cada año que pasa,
# la expectativa de vida aumenta ~0.23 años
# (casi 3 meses)

Correlación: 0.998

Relación casi perfecta. El modelo explica el 99.5% de la variabilidad (R² = 0.995).

Tendencia lineal positiva (1952-2007)

Interpretando Significancia Estadística

Símbolo Significado Confianza Interpretación
*** p < 0.001 99.9% Relación extremadamente significativa
** p < 0.01 99% Muy significativa (publicable)
* p < 0.05 95% Significativa (umbral estándar)
. p < 0.1 90% Marginalmente significativa
(vacío) p > 0.1 < 90% No significativa (descartar)

⚠️ Nota: El umbral del 95% es convención arbitraria. Contexto y conocimiento de dominio siempre prevalecen sobre la significancia estadística.

El "dónde" es central en políticas públicas. Los mapas son herramientas de visualización esenciales para entender dinámicas territoriales y desigualdades espaciales.

Datos Vectoriales

  • Líneas: Rutas de transporte, avenidas
